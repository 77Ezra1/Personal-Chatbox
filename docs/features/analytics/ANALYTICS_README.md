## 🐛 故障排除

### 常见问题

#### Q: 费用显示为$0.0000？
```
A: 可能原因:
   - 还没有发送消息
   - Token数据未同步

解决方法:
   - 发送几条消息
   - 刷新数据面板
```

#### Q: 货币设置后没更新？
```
A: 解决方法:
   - 点击刷新按钮
   - 或刷新浏览器页面
```

#### Q: API调用次数和消息数不一致？
```
A: 这是正常的！
   - 消息数包括你的提问和AI回复
   - API调用只计算AI回复
```
